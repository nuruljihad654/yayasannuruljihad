1) Buka VS Code -> File > Open Folder -> pilih folder project-ponpes-nuruljihad
2) Pastikan semua file di folder img/ sudah tersedia:
   - logo-nuruljihad.png (gunakan file logo resmi yang Anda upload)
   - hero.jpg (banner), struktur.png (bagan), operator.jpg, pengurus*.jpg, guru*.jpg
   - placeholder.png (opsional)
3) Buka index.html di browser (klik kanan > Open with Live Server / atau double-click)
4) Sesuaikan konten teks, nomor telp, video ID, dan nama kepala sekolah sesuai data nyata
5) Untuk form kontak produksi: hubungkan ke endpoint server / Google Forms / layanan email.
